サービス終了

見てるなう(仮)は Twitter API の無償提供終了に伴い、サービスを終了いたしました。

2012 年 4 月 28 日からおよそ 11 年の長きにわたりご利用くださり本当にありがとうございました。

なお、見てるなう(仮)が保存していたアクセストークンは削除いたしましたが、ご利用になられていた皆さまにおきましても Twitter の設定ページよりアクセス権を取り消すことができます。

見てるなう(仮)

あなたが今見ているウェブページをTwitterで共有するためのサービス

ご利用方法

  1. Twitter でログイン
  2. [見てるなう]リンクをブックマークバーにドラッグアンドドロップ
  3. 好きなページでブックマークバーの[見てるなう]をクリック
  4. コメントを入力して Twitter に投稿

投稿例(コメント無し)

Google: ux.nu/DVKyN #miteru

— よしださん (@yosida95) 5月 14, 2012

投稿例(コメント有り)

(コメント) - Google: ux.nu/DVKyN #miteru

— よしださん (@yosida95) 5月 14, 2012
